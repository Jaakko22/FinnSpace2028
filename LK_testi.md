# Md testailua

### REST perusajatus
REST on APIen toteuttamiseen tarkoitettu arkkitehtuurimalli.
Se määrittelee sovellukset tietoa käsittelevien osien, tietokohteiden,
sekä näiden yhteyksien kautta.

### Mitä CRUD tarkoittaa
- CRUD muodostuu:
  Kirjain | tarkoittaa
  ------- | ----------
  C | Change 
  R | Read
  U | Update
  D | Delete
